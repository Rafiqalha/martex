# TumbuhSehat — Aplikasi Pantau Tumbuh Kembang Anak

TumbuhSehat adalah aplikasi Android yang membantu orang tua memantau **pertumbuhan dan status gizi anak secara mudah, cepat, dan akurat**.

Dirancang untuk digunakan sehari-hari tanpa perlu memahami istilah medis seperti z-score atau kurva WHO.

---

## Tujuan Aplikasi

Membantu orang tua menjawab 2 pertanyaan penting:

1. **Apakah pertumbuhan anak saya normal?**
2. **Apa yang harus saya lakukan hari ini?**

---

## Fitur Utama

### 1. Cek Status Pertumbuhan

* Input: umur, tinggi badan, berat badan
* Hasil langsung:

  * Normal
  * Risiko stunting
  * Stunting

✔ Tanpa perlu memahami grafik WHO
✔ Hasil ditampilkan dalam bentuk sederhana dan jelas

---

### 2. Rekomendasi Nutrisi

* Saran makanan sesuai kondisi anak
* Fokus pada bahan lokal & terjangkau
* Praktis untuk diterapkan sehari-hari

---

### 3. Riwayat Pertumbuhan

* Simpan data pengukuran sebelumnya
* Pantau perubahan dari waktu ke waktu
* Membantu melihat tren pertumbuhan anak

---

### 4. Offline Mode (Tanpa Internet)

* Aplikasi tetap bisa digunakan tanpa koneksi
* Data tersimpan aman di perangkat
* Akan otomatis sinkron saat ada internet

✔ Cocok untuk daerah dengan sinyal terbatas

---

### 5. Mode Kader (Opsional)

* Input data banyak anak sekaligus
* Digunakan saat kegiatan Posyandu

---

## Cara Kerja

1. Buka aplikasi
2. Masukkan data anak
3. Sistem menghitung status pertumbuhan
4. Hasil langsung ditampilkan
5. Ikuti rekomendasi yang diberikan

---

## Privasi Data

Kami menjaga data pengguna dengan ketat:

* Tidak menyimpan:

  * Nama lengkap
  * NIK
  * Alamat detail
* Data hanya digunakan untuk analisis pertumbuhan
* Data tersimpan aman di perangkat dan server

## Kenapa TumbuhSehat Penting?

Banyak orang tua tidak menyadari masalah pertumbuhan anak karena:

* Kurangnya pemahaman grafik kesehatan
* Akses ke layanan kesehatan terbatas

TumbuhSehat hadir untuk memberikan **informasi yang sederhana, cepat, dan bisa langsung digunakan**.
